Bootstrap: 
          - it is html,css and js framework
          - which is use to make web app attractive,responsive and mobile friendly
          - it provide inbuilt classes
          - latest version of Bootstrap is Bootstrap5

* there are 2 ways to use Bootstrap
1. by using CDN(Content Delivery Network) link
2. by downloading

*official website Bootstrap is
getbootstrap.com 

*how to copy and use CDN link

*heading classes: .display-1 to .display-6  we should use inside the class 

.container : it will apply 15px margin from left and right of an element
.container-fluid : it will take full width of screen

note : above 2 classes is used to wrap element or Content

                 1. row class
                 2. col class
